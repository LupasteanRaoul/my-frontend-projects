About This Portfolio

Why Vanilla JavaScript?

All projects are built using only vanilla JavaScript to demonstrate core web development skills without relying on frameworks. This approach ensures:

  • Better understanding of JavaScript fundamentals
  • Faster loading times and better performance
  • Smaller bundle sizes and no dependencies
  • Improved debugging and problem-solving skills
